# Mini Game Collection

A collection of mini games for Minecraft Fabric.

## Features

- **Game Mode Switcher**: Automatically switches players' game modes every configurable interval
- **Configurable**: Uses Cloth Config API for easy configuration
- **Localization**: Supports English and Chinese
- **Commands**: Provides commands to control game features

## Installation

1. Install Fabric Loader
2. Install Fabric API
3. Install Cloth Config API
4. Install ModMenu (optional, for configuration GUI)
5. Put the Mini Game Collection mod jar in your mods folder

## Commands

- `/minigc gamemode enable` - Enable game mode switching
- `/minigc gamemode disable` - Disable game mode switching
- `/minigc gamemode interval <seconds>` - Set game mode switch interval
- `/minigc status` - Check current status

## Configuration

Configuration file is located at `.minecraft/config/minigc/minigc-common.properties`
